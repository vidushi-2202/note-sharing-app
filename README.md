# Notes Sharing Platform

A full-stack Notes Sharing Platform developed using PHP, HTML, CSS, JavaScript, and MySQL.  
The platform allows users to upload, manage, search, and share study notes through a responsive and user-friendly interface.

---

## 🚀 Features

- User Authentication System
- Upload and Share Notes
- Search Notes Functionality
- Bookmark Notes
- Rating and Comment System
- Notifications Feature
- Admin Dashboard
- Manage Users and Subjects
- Report Management System
- Responsive UI Design
